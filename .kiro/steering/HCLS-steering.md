this project is a self contained demo of AWS Clean Rooms custom ML, implemented for a financial services fraud use case, with financial services data from a Bank and from a payment processor. 
there's also a quicksight dashboard that shows some key financial services metrics.
read README.md for project context.

Objective of this project is to find an healtcare and life sciences (HCLS) use case (searching on internet), identify two fictional parties (a pharma company and another HCLS company) and their respective datasets (list of columns with join key).
then we need to change the data generation scripts to the new HCLS datasets
then we need to change the ML models to new feature and inference
then we need to change the quicksight dasboard to show HCLS metrics
